# 📚 Student Management System (JDBC + MySQL)

A simple **Java JDBC-based CRUD application** for managing student records using MySQL database.  
This project demonstrates core backend development concepts like JDBC, DAO pattern, and database operations.

---

## 🚀 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- ❌ Delete Student

---

## 🛠️ Tech Stack

- ☕ Java
- 🗄️ JDBC
- 🐬 MySQL
- 🧑‍💻 Eclipse / IntelliJ
